# AGENTS.md: 基本

このreferenceを読むときは、このファイルだけで判断せず、関連するreferencesも必ず読むこと。

## ルール

- 曖昧な確認、推測は常に禁止。
- 必ずファイルでtodoを管理すること。
  - コンテキストウィンドウを信用しない。確実なソリューションを使用する。
- コードを参照していない場合は、通常の受け答えを行う。
- すべての回答は日本語で行う。
- 長々と語らず、簡潔に説明すること。
  - たとえば、~はこうです。~ならこうです。ですが、~ならこうです。などとくどくど話さない。
- テストのコードを参照している場合は、以下の手順に従う。
  - エラーメッセージがを添付された場合は、そのエラーメッセージに該当する箇所を指摘する。
  - エラーメッセージが添付されていない場合は、テストの実装に必要なコードを提案する。
